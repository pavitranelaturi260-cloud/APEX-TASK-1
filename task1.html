Start with HTML

1. HTML Basics

HTML gives the page its structure. Below we use headings, paragraphs, images, and links.

Headings & Paragraphs

Use <h1>...</h1> for the main title and <p> for text blocks. Keep one <h1> per page.

Images

Add visuals with <img src="..." alt="..." />. The alt text helps with accessibility and SEO.

Links

Create navigation using <a href="#">. External links should use target="_blank" and rel="noreferrer".
